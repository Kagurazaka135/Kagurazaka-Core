# memory.py — 记性

**让 AI 记得"刚才说了什么"。** 不是那种高大上的向量数据库 RAG，就是最朴素的——拿本子记下来，回头翻。

## 三层记忆 + 一个缓存

```
工作记忆（最近 7 条）  ← 当前会话的关键信息，满了就踢最早的
会话历史（最近 10 轮）  ← 每轮对话的完整记录
长期记忆（存磁盘）    ← 重要的东西写进 JSON 文件，下次启动还在
搜索缓存（最近 20 条）  ← 搜过的结果存 30 分钟，下次搜同样的词直接用
```

## 怎么用

### 上下文增强
你说"刚才那个再改改"——它从历史里翻出上一轮的内容拼进去。靠正则匹配"刚才/刚刚/这个/那个/继续"这些指代词。

### 搜索缓存
搜之前先翻缓存：关键词匹配 → 模糊匹配 → 过期了就不要。30 分钟 TTL。

### 长期记忆
超过 200 字的输出，或者不是闲聊类型的任务，自动存进长期记忆。按 chat/code/logic 分类存，每类有上限。

## 中文关键词提取

用 2~6 字中文短语切分，去掉"的""了""是""这个"之类没信息量的虚词（100+ 个），按频率取 top 5。用来做话题关联和缓存匹配。
