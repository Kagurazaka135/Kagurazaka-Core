# logger.py — 会话日志 & 统计

**每次 LLM 调用都记，退出时算统计。**

## 每条日志记什么

```json
{
    "session": "session_1716123456",
    "timestamp": "2026-05-19T14:00:01",
    "node": "classifier",
    "call_type": "llm",
    "input_len": 240,
    "output_len": 8,
    "latency_ms": 1234.5,
    "success": true,
    "error": ""
}
```

## 都有谁在记

`core.py` 的每个步骤都调 `log.record()`：

```
classifier   →  记 1 条（搜索判断）
parser       →  记 1 条（分类解析）
chat         →  记 1 条（如果有）
code         →  记 1 条（如果有）
logic        →  记 1 条（如果有）
polish       →  记 1 条（润色，重试时可能多条）
```

## 退出时展示什么

```
============================ 会话统计 ============================
  会话ID:     session_1716123456
  总调用数:   5
  成功:       5  |  失败: 0
  平均延迟:   2100ms  |  最长: 4500ms
  会话耗时:   12.3s
----------------------------------------
  [classifier] 1次 | 成功1 | 平均1200ms
  [parser]   1次 | 成功1 | 平均2800ms
  [chat]     1次 | 成功1 | 平均1500ms
  [polish]   1次 | 成功1 | 平均1800ms
  [code]     0次
  [logic]    0次
=================================================================
```

## 历史日志

`python main.py --stats` 从 `logs/` 目录读所有 `*.jsonl` 文件，按会话聚合统计。

```bash
历史统计 - 5 个日志文件, 12 个会话

  [session_xxx] 6次调用 | 成功率 100% | 平均 2200ms
  [session_yyy] 4次调用 | 成功率 75%  | 平均 3100ms
```
